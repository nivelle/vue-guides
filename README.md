# 非前端程序员的VUE教程

### 项目初始化(本项目使用idel初始化,使用自定义的运行环境)

```
 
 npm install -g cnpm --registry=https://registry.npm.taobao.org //使用淘宝镜像
 
 cnpm install;//安装npm
 
 npm run dev;//试运行
 
```

### 使用elementUI来保持样式,在man.js全局引入

```

cnpm i element-ui -S

```
